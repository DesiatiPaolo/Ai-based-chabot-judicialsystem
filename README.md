# Ai-based-judicial-chatbot
Judicial Chatbot using RASA

## Setup
- install pandas
```
python -m pip install pandas
```


##  Running the chatbot
- Run rasa action server on separate terminal
```
rasa run actions
```
- Test bot on shell
```
rasa shell
```
